Informations

ShakeMap is a jQuery data-manager plugin using with Google Maps.

It's designed for manage and elaborate thousand of places in a single map.

It's based on jmapping plugin created by Brian Landau of Viget Labs.

Shakemap is builded around a powerfull plugin for clustering, Marker Clusterer.

You can easily manage and display a very large amount of markers in a single map, drastically increase the performace of the page.

Working, also, with Overlapping Marker Spiderfier, a simple yet elegant way to show more marker on the same place.

You can, even, customize the appearance of the info window through Infobox plugin.

Finally, you can use the powerfull of HTML5 and the Geolocation.

If the browser support geolocation feature, you can show the position of the user in the map and realaborate the data as you wish!
